# Logic Pro ファイル名クリーナー

Logic Proで書き出したWAVファイル名の末尾に付く`_1`などの番号や`_bip`を取り除く、macOS Finder用のクイックアクションです。

FinderでWAVファイル、またはWAVファイルが入ったフォルダを選んで実行すると、対象ファイルをその場でリネームします。

## できること

- `Vocal_bip.wav`を`Vocal.wav`に変更
- `Guitar_1.wav`を`Guitar.wav`に変更
- `Bass_bip_2.wav`を`Bass.wav`に変更
- フォルダを選んだ場合は、その中の`.wav`ファイルを再帰的に処理
- リネーム後の名前が同じフォルダ内ですでに存在する場合は、そのファイルをスキップ

## 注意

このworkflowはファイル名を直接変更します。初めて使うときは、バックアップ済みのファイルやテスト用フォルダで動作を確認してください。

対象は拡張子が小文字の`.wav`のファイルです。`.aif`、`.mp3`、`.WAV`などは処理しません。

## インストール

1. このリポジトリをダウンロードします。
2. `Logic Pro ファイル名クリーナー.workflow`を`~/Library/Services/`にコピーします。
3. Finderを開き、WAVファイルをControlクリックします。
4. `クイックアクション`、または`サービス`から`Logic Pro ファイル名クリーナー`を選びます。

`~/Library/Services/`が存在しない場合は作成してください。

```sh
mkdir -p ~/Library/Services
cp -R "Logic Pro ファイル名クリーナー.workflow" ~/Library/Services/
```

表示されない場合は、Finderを再起動するか、ログアウトしてから再ログインしてください。

## 使い方

Finderで処理したいWAVファイル、またはWAVファイルが入ったフォルダを選択します。

Controlクリック、または右クリックでメニューを開き、`クイックアクション`から`Logic Pro ファイル名クリーナー`を選びます。メニューバーの`Finder`、`サービス`から実行できる場合もあります。

複数ファイルを選択してまとめて実行できます。フォルダを選んだ場合は、そのフォルダ以下にある`.wav`ファイルをまとめて処理します。

## ショートカットキーの設定

よく使う場合は、クイックアクションにキーボードショートカットを割り当てると便利です。

1. macOSの`システム設定`を開きます。
2. `キーボード`を開き、`キーボードショートカット`をクリックします。
3. 左側の一覧から`サービス`を選びます。
4. `ファイルとフォルダ`内にある`Logic Pro ファイル名クリーナー`を探します。
5. 右側の`なし`または現在のショートカット欄をダブルクリックします。
6. 割り当てたいキーを押します。

例として、作者は`Command + Control + L`を設定しています。

ショートカットが効かない場合は、ほかのアプリやmacOSのショートカットと競合している可能性があります。別の組み合わせに変更してください。

## 仕組み

workflowはAutomatorの`シェルスクリプトを実行`アクションで作られています。Finderから渡されたファイルやフォルダを受け取り、`.wav`ファイル名の拡張子を除いた部分に対して、末尾の`_bip`と`_数字`を削除します。

リネーム後のファイル名がすでに存在する場合は、上書きせずにスキップします。
